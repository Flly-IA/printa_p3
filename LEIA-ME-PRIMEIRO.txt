┌─────────────────────────────────────────────────────────────────────────┐
│  🎯 O QUE É ISSO?                                                        │
└─────────────────────────────────────────────────────────────────────────┘

API REST completa que transforma seu script Python local em um serviço web
profissional para geração automática de cardápios em PDF/PNG/CDR usando
CorelDRAW.

│  📦 CONTEÚDO DO PACOTE                                                   │
└─────────────────────────────────────────────────────────────────────────┘

  • build_cardapio_dinamico.py   → Seu código original (já incluído)

│  ✅ REQUISITOS                                                           │

  • Python 3.8 ou superior

┌─────────────────────────────────────────────────────────────────────────┐
│  🔌 EXEMPLO DE USO                                                       │
└─────────────────────────────────────────────────────────────────────────┘

  # Via cURL
  curl -X POST "http://localhost:8000/cardapio/gerar" \
    -F "file=@teste_input.txt" \
    -F "font=Arial" \
    -F "font_size=10"

  # Via Python
  import requests
  with open('teste_input.txt', 'rb') as f:
      r = requests.post('http://localhost:8000/cardapio/gerar',
                        files={'file': f})
  print(r.json())


┌─────────────────────────────────────────────────────────────────────────┐
│  🌐 DEPLOY EM PRODUÇÃO                                                   │
└─────────────────────────────────────────────────────────────────────────┘

  Azure VM / AWS EC2 / DigitalOcean:

  1. Criar VM Windows Server 2019/2022
  2. Instalar Python + CorelDRAW
  3. Clonar repositório
  4. Executar: deploy_windows.bat
  5. Configurar como serviço Windows (NSSM)
  6. Abrir porta 8000 no firewall
  7. Opcional: Nginx reverse proxy + HTTPS

  Detalhes completos em README_API.md


┌─────────────────────────────────────────────────────────────────────────┐
│  🆘 PROBLEMAS COMUNS                                                     │
└─────────────────────────────────────────────────────────────────────────┘

  ❌ CorelDRAW não inicializa
     → Execute como Administrador

  ❌ Templates não encontrados
     → python create_templates.py

  ❌ Porta 8000 ocupada
     → Altere porta em start_api.py

  ❌ Módulo não encontrado
     → pip install -r requirements_api.txt

  🔍 Veja TROUBLESHOOTING.md para mais soluções
